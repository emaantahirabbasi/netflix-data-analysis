# 🎬 Netflix Data Analysis

## Overview
Exploratory data analysis on a real-world Netflix dataset 
(7,787 titles) uncovering content trends, genre patterns, 
and duration changes from 1980 to 2020.

## 📊 Key Insights
- 27% decline in average movie duration from 1980–2020
- Drama is the #1 genre with 1,384 titles
- Movies outnumber TV Shows 2:1 on the platform
- US, India, UK are top content-producing countries
- Peak content addition years identified

## 🛠️ Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn

## 📁 Dataset
Netflix Movies and TV Shows — [Kaggle Link](https://www.kaggle.com/datasets/shivamb/netflix-shows)
